"Pouce d'Or" RESTful service (Symfony)
======================================

Repository for the symfony version of restful service for Pouce d'Or Website : www.poucedor.fr

## Major bundles used

* Symfony 3.1
* Doctrine
* phpUnit
* FOSUserBundle
* FOSOauthServerBundle
* FOSRestBundle
* JMSSerializerBundle
* NelmioApiDocBundle
